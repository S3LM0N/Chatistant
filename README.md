사용자와 대화하고 제한적이지만 규칙기반의 행동까지 가능한 챗봇


##  Summary

다소 문제가 많았던 전 프로젝트에 리팩터링과 최적화, 일부 기능은 디벨롭을 거치는 중입니다.

현재 Transformer 와 Ko-GPT 를 비교하는 중입니다. 

따라서 지금은 오로지 규칙기반의 명령어만 실행이 가능합니다.
